# 股票交易 Demo 演示流程

## 前置准备

```bash
cd ~/stock-trading-demo
npm install
npm run build
npm run start   # 启动服务，默认 http://localhost:3000
```

---

## 桌面端演示路径

### Step 1: 首页查看市场

- **URL:** http://localhost:3000/
- **操作:** 打开首页
- **预期结果:**
  - 显示暗色金融主题界面，glass morphism 玻璃拟态风格
  - 顶部展示市场概览（上证/深证/创业板指数）
  - 板块列表以卡片形式展示，含涨跌幅数据
  - 热门股票排行
- **验证点:** 界面风格为深色主题，卡片有毛玻璃效果，数据加载正常

### Step 2: 点击热门板块 — 光通信

- **URL:** http://localhost:3000/sectors/1
- **操作:** 在首页板块列表中点击「光通信」板块卡片
- **预期结果:**
  - 页面跳转至板块详情页
  - 显示板块名称、涨跌幅、成交量等信息
  - 下方展示成分股列表（股票代码、名称、价格、涨跌幅）
- **验证点:** 板块数据与首页一致，成分股列表可滚动

### Step 3: 查看板块详情和成分股

- **URL:** http://localhost:3000/sectors/1
- **操作:** 浏览成分股列表
- **预期结果:**
  - 成分股按涨跌幅排序
  - 每只股票显示代码、名称、最新价、涨跌幅
  - 点击任意股票可进入详情页
- **验证点:** 列表数据完整，排序合理，点击响应正常

### Step 4: 点击中际旭创 (300308)

- **URL:** http://localhost:3000/stocks/300308
- **操作:** 在成分股列表中点击「中际旭创」
- **预期结果:**
  - 进入股票详情页
  - 显示股票基本信息（名称、代码、行业）
  - 展示 K 线图（默认蜡烛图）
  - 右侧或下方显示成交量柱状图
  - 底部有买入/卖出操作区
- **验证点:** 股票信息正确，K线图正常渲染

### Step 5: 切换 K 线图类型

- **URL:** http://localhost:3000/stocks/300308
- **操作:** 点击图表类型切换按钮，依次切换：蜡烛图 → 面积图 → 折线图 → OHLC
- **预期结果:**
  - **蜡烛图 (Candlestick):** 标准红绿蜡烛，带上下影线
  - **面积图 (Area):** 填充区域图，展示价格趋势
  - **折线图 (Line):** 简洁折线连接收盘价
  - **OHLC:** 开高低收柱状图
  - 切换时有过渡动画
- **验证点:** 四种图表均正确渲染，切换流畅

### Step 6: 点击买入

- **URL:** http://localhost:3000/stocks/300308
- **操作:** 在交易面板点击「买入」按钮
- **预期结果:**
  - 交易面板切换至买入模式
  - 默认填入当前价格
  - 可调整价格和数量
  - 显示预估金额
- **验证点:** 买入面板激活，价格/数量输入框可用

### Step 7: 提交模拟交易

- **操作:** 调整数量后点击「提交」/「买入」确认按钮
- **预期结果:**
  - 显示提交确认动画
  - 模拟撮合过程（短暂等待）
  - 状态从「已提交」→「正在撮合」→「已成交」
- **验证点:** 动画流畅，状态变化自然

### Step 8: 查看模拟成交反馈

- **操作:** 查看成交结果弹窗或跳转至委托页面
- **URL:** http://localhost:3000/orders
- **预期结果:**
  - 显示成交详情：股票、方向、价格、数量、金额
  - 委托列表中新增一条记录
  - 状态标记为「已成交」
- **验证点:** 成交数据与提交一致，列表正确更新

## 移动端演示路径

### Mobile Step 1: 移动端首页

- **URL:** http://localhost:3000/mobile
- **操作:** 打开移动端首页
- **预期结果:**
  - 响应式布局适配小屏幕
  - 板块列表紧凑展示
  - 触摸交互友好
- **验证点:** 布局无溢出，文字可读

### Mobile Step 2: 板块详情

- **URL:** http://localhost:3000/mobile/sectors/1
- **操作:** 点击「光通信」板块
- **预期结果:**
  - 移动端板块详情页
  - 成分股列表适配小屏
- **验证点:** 列表正常滚动，点击进入股票

### Mobile Step 3: 交易页面

- **URL:** http://localhost:3000/mobile/trade/300308
- **操作:** 点击中际旭创进入交易页
- **预期结果:**
  - K 线图自适应屏幕宽度
  - 交易操作区在图表下方
  - 买入/卖出切换方便
- **验证点:** 图表不溢出，操作区触控区域足够大

### Mobile Step 4: 模拟交易

- **操作:** 输入数量，点击买入确认
- **预期结果:**
  - 模拟成交流程
  - 成功反馈展示
- **验证点:** 全流程在移动端完成

## 路由总览

| 页面 | 路由 | 说明 |
|------|------|------|
| 桌面首页 | `/` | 市场概览 + 板块列表 |
| 板块详情 | `/sectors/[sectorId]` | 板块信息 + 成分股 |
| 股票详情 | `/stocks/[stockCode]` | K线图 + 交易面板 |
| 委托记录 | `/orders` | 历史委托列表 |
| 移动首页 | `/mobile` | 移动端市场概览 |
| 移动板块 | `/mobile/sectors/[sectorId]` | 移动端板块详情 |
| 移动交易 | `/mobile/trade/[stockCode]` | 移动端交易页 |
