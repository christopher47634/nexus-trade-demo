# P2 验收标准

## P2-B 验收标准

### 1. visual-lab 实验页

| 检查项 | 标准 |
|--------|------|
| `/visual-lab` 可访问 | HTTP 200 |
| 3 张实验卡片渲染 | 无空白/错位 |
| CSS/Canvas 版本切换 | 点击后正确切换 |
| FPS 计数器显示 | 实时更新 |
| 移动端单列布局 | 390px 宽度无横向溢出 |
| build 通过 | npm run build 无错误 |
| 不影响主页 | `/` 行为不变 |

### 2. 高级板块卡片

| 检查项 | 标准 |
|--------|------|
| 视觉风格 | 保持金融终端感，不花哨 |
| Canvas FPS | ≥ 55 FPS |
| CSS 动画流畅度 | 无卡顿/闪烁 |
| hover 效果 | scale + glow 正常 |
| 数据展示 | 涨跌幅/成交额/资金流向仍可见 |
| 点击功能 | 仍可跳转板块详情 |
| 移动端适配 | 单列布局，动画正常 |

### 3. hover 光晕

| 检查项 | 标准 |
|--------|------|
| 光晕跟随鼠标 | 实时跟踪，无延迟 |
| 性能 | FPS ≥ 55 |
| 淡入淡出 | 鼠标进入/离开时平滑 |
| 移动端禁用 | 移动端无光晕、无报错 |
| 不影响点击 | 卡片仍可正常点击 |

### 4. shared layout transition

| 检查项 | 标准 |
|--------|------|
| 板块卡片 → 详情 | 标题/涨跌幅平滑飞入 |
| 详情 → 返回 | 反向动画正常 |
| 无闪烁 | 转场过程中无白屏/跳帧 |
| 无错位 | 元素落点正确 |
| 不影响功能 | 详情页所有功能正常 |

## P2-B 性能红线

| 指标 | 红线 |
|------|------|
| 首屏 LCP | > 2.5s 不通过 |
| 首页 JS | > 250KB 不通过 |
| Canvas FPS | < 45 不通过 |
| 内存 | > 120MB 不通过 |

## P2-B 回退标准

任何增强点触发红线 → 设环境变量为 false → 回退到 V1 行为 → 不阻塞其他功能。

- 资产卡片反射：反射效果可见，不影响数据阅读
- 数字滚动：资产数字有滚动动画，最终值正确
- 更多板块卡片：视觉风格与前 3 张一致
