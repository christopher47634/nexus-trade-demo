# Sector Visual Identity System

## 视觉语法（所有板块统一）

```
Layer 0: 深色玻璃底        — 深黑蓝，不要纯黑
Layer 1: 行业主视觉水印     — 低透明度线稿，不抢文字
Layer 2: 精密线条 / 数据流   — 0.5-1px，opacity 0.06-0.18
Layer 3: 动态点 / 脉冲 / 扫描 — 数量少，速度慢，有行业意义
Layer 4: Hover Glow         — 局部响应，增强 10-20%
```

## 动效参数规范

| 参数 | 值 |
|------|-----|
| idle 动画周期 | 8-16s |
| hover 淡入 | 450-700ms |
| hover 淡出 | 600-900ms |
| 粒子数量 | 3-8 个/卡片 |
| 主视觉 opacity | 0.08-0.22 |
| 高光 opacity | 0.10-0.18 |
| glow 最大 opacity | 0.12-0.16 |
| 线条宽度 | 0.5-1px |

## 统一材质

- 所有卡片共享：深黑蓝玻璃底、微边框光、相同圆角和 padding
- 线条风格：精密、克制、1px 为主
- 色彩体系：每个板块有 1 主色 + 1 辅色，不允许霓虹色
- 字体层级：板块名 > 涨跌幅 > 副标题 > 数据

## 实施优先级

### 第一批（最成熟 + 最有辨识度）

1. 光通信 — 已有，精修
2. 算力 — 已有，精修

4. 新能源 — 新做（能量流方向明确）
5. 机器人 — 新做（机械臂轨迹辨识度高）

### 第二批

8. 军工 — 新做

### 第三批

9. 矿山 — 新做
10. 医药 — 新做

## 技术决策

- 10 个板块全部用 Canvas 2D，白酒可加 SVG 酒瓶轮廓
- 不用 Three.js / GLSL
- 不用 AI 生成图片
- 不用大图背景
- 所有动效共享同一套参数规范
