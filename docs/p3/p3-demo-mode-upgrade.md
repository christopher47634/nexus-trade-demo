# P3 Demo Mode Upgrade

## 当前状态

Demo Mode 是一个 8 步引导链路：
0. 首页光通信卡片高亮
1. 板块详情中际旭创高亮
2. K线图切换器高亮
3. 买入按钮高亮
4. 输入区高亮
5. 确认买入高亮
6. 查看订单高亮
7. 订单表格高亮

每步有文字说明、金色虚线边框、可跳过。

## 问题

1. **像新手引导，不像产品演示**
   - 当前文案是操作指引（"点击这里"、"输入金额"）
   - 应该是价值展示（"看，这就是我们的K线系统"）

2. **没有演示感**
   - 没有自动播放
   - 没有进度指示
   - 没有演示路线面板

3. **没有录屏友好模式**
   - 调试内容（Demo标签、FPS）没有隐藏
   - 没有全屏模式

## 升级方案

### 方案 1：文案升级（0.5 天）

把操作指引改为价值展示：

| 步骤 | 当前文案 | 建议文案 |
|------|----------|----------|
| 0 | "点击光通信板块查看详情" | "10个行业板块，每个都有独立视觉身份" |
| 1 | "点击查看中际旭创详情" | "实时行情数据，深度个股分析" |
| 2 | "切换K线图类型" | "日K、周K、月K一键切换" |
| 3 | "点击买入按钮" | "模拟交易，零风险体验完整交易链路" |
| 4 | "输入购买数量和价格" | "精确控制交易参数" |
| 5 | "确认买入" | "一键成交，毫秒级响应" |
| 6 | "查看订单记录" | "完整交易历史，localStorage持久化" |
| 7 | "订单详情" | "成交价、数量、时间，一目了然" |

### 方案 2：Presentation Mode（1-2 天）

新增一个演示模式：
- 自动播放（每步 4-5 秒）
- 底部显示进度条 + 当前步骤名
- 左右箭头可手动切换
- 空格暂停/继续
- 全屏模式（隐藏浏览器地址栏提示）
- 录屏模式（隐藏所有调试UI）

入口：
- 首页右上角"演示"按钮
- URL参数 `?presentation=true`

### 方案 3：演示路线面板（1 天）

在Demo Mode中增加一个浮动面板：
```
┌─────────────────────────┐
│ ● 模拟交易演示          │
│ ○───○───○───●───○───○   │
│ 步骤 4/8: 输入交易参数   │
│ [上一步] [下一步] [跳过] │
└─────────────────────────┘
```

显示：
- 当前步骤 / 总步骤
- 步骤名称
- 进度圆点
- 控制按钮

## 建议

**P3-B**：方案 1 文案升级（0.5 天，零风险）
**P3-C**：方案 2 + 3（2 天，产品级提升）
