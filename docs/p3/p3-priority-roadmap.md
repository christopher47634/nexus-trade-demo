# P3 Priority Roadmap

## P3-B：最小高价值升级（预计 1 天）

低风险，可直接上线。

| 任务 | 预计时间 | 说明 |
|------|----------|------|
| HoverGlow 惯性升级 | 0.5天 | lerp 动态范围、velocity 影响、soft falloff |
| Demo Mode 文案升级 | 0.5天 | 操作指引→价值展示 |

**总计**：1 天
**风险**：极低
**上线**：直接上线 Production

---

## P3-C：视觉交互增强（预计 3-5 天）

中风险，需性能测试。

| 任务 | 预计时间 | 说明 |
|------|----------|------|
| 卡片 Canvas 视差响应 | 1天 | mouseX/Y 影响 Canvas 内元素偏移 2-5px |
| 行业视觉 hover 局部响应 | 1天 | 光点加速、芯片亮起、雷达锁定等 |
| 首页→板块详情转场升级 | 1天 | Canvas scale-up、图标飞入、边框 morph |
| 板块详情→个股详情转场 | 0.5天 | 股票名+涨跌幅 layoutId |
| 演示路线面板 | 0.5天 | 浮动进度面板 + 自动播放 |

**总计**：4 天
**风险**：中（需测试每帧 mouse transform 性能）
**上线**：需 QA 后上线

---

## P3-D：Awwwards 实验页（预计 4-5 天）

高风险，只放实验页。

| 任务 | 预计时间 | 说明 |
|------|----------|------|
| 全局 mouse shader / liquid light | 2天 | Canvas overlay + 粒子系统 |
| 卡片边缘折射效果 | 1天 | glass refraction 模拟 |
| 自定义光标样式 | 0.5天 | dot + ring + trail |
| /visual-lab 实验页升级 | 1天 | 所有实验效果集中展示 |

**总计**：4.5 天
**风险**：高（全局 overlay + shader 性能）
**上线**：❌ 只放 /visual-lab 或 landing hero

---

## 依赖关系

```
P3-B（1天）→ 可独立上线
    ↓
P3-C（4天）→ 依赖 P3-B 的 HoverGlow 基础
    ↓
P3-D（4.5天）→ 独立实验，不影响主线
```

## 不做的

| 项目 | 原因 |
|------|------|
| Three.js 3D 场景 | 成本高，Demo 项目不需要 |
| GLSL 全站 shader | 性能风险大，只放实验页 |
| 真实行情接入 | 不是 P3 范围 |
| 券商 API | 不是 P3 范围 |
| 登录注册 | 不是 P3 范围 |
| 主题系统 | 可以做但优先级低 |

## 总时间线

| 阶段 | 时间 | 产出 |
|------|------|------|
| P3-B | 1 天 | HoverGlow 惯性 + Demo 文案 |
| P3-C | 4 天 | cursor interaction + 转场 + 演示面板 |
| P3-D | 4.5 天 | Awwwards 实验页 |
| **总计** | **9.5 天** | |
