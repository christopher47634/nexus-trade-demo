## 拒绝机制架构

```
用户输入 → [UI 层实时校验] → 内联警告 (即时反馈)
                    ↓
          点击确认 → [TradePanel 层]
                    ↓
          点击确认委托 → [引擎层 executeSell/executeBuy]
                    ↓
              校验失败 → 返回 { success: false, error: "..." }
                    ↓
          UI 显示错误消息，step 回退到 "input"
                    ↓
              持仓/资金/订单/流水均不变
```
