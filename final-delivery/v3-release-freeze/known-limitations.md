# V3 Known Limitations

## 已知限制（不做修复，仅记录）

### 1. 非真实行情数据

- 所有股票价格、涨跌幅、成交额、资金流均为 mock 数据
- K 线图使用随机生成的历史数据
- 不连接任何真实行情源
- **影响**: 无法用于真实交易决策

### 2. 非真实交易

- 买入/卖出操作仅写入 localStorage
- 不连接任何券商 API
- 订单数据仅在当前浏览器有效
- **影响**: 清除浏览器数据后订单丢失

### 3. 无用户系统

- 无登录/注册
- 无账户隔离
- Demo Mode 使用 localStorage 持久化 step
- **影响**: 多人共用同一浏览器会互相干扰

### 4. Canvas 视觉为装饰性

- Canvas 动效不承载真实数据
- 光通信的"光点"不对应真实网络流量
- 算力的"脉冲"不对应真实算力指标
- **影响**: 视觉效果与业务数据脱钩

### 5. 行业覆盖有限

- 当前仅 10 个行业板块
- 未覆盖 A 股所有行业分类
- 未支持自定义添加板块
- **影响**: 无法反映完整市场

### 6. 移动端 Canvas 降级

- 移动端 Canvas 使用更小尺寸和更低帧率
- HoverGlow 在移动端完全禁用
- 部分动效细节在小屏幕上不可见
- **影响**: 移动端视觉体验弱于 Desktop

### 7. 无实时推送

- 页面不使用 WebSocket
- 数据不自动刷新
- 需手动刷新页面获取"新"数据
- **影响**: 无法模拟实时交易体验

### 8. 浏览器兼容性

- Canvas 2D 需要现代浏览器
- backdrop-filter 在部分旧浏览器不支持
- prefers-reduced-motion 需要浏览器支持
- **影响**: IE11 及以下不可用

### 9. 性能边界

- 10 张 Canvas 同时渲染在低端设备可能掉帧
- DPR 上限 2，4K 屏幕不会额外渲染
- IntersectionObserver 懒加载，但首次进入视口有初始化开销
- **影响**: 低端手机可能出现短暂卡顿

### 10. 数据持久化局限

- 订单存储在 localStorage，key: `nexus-trade-orders`
- Demo Mode step 存储在 localStorage
- 无服务端持久化
- **影响**: 换设备/清缓存后数据丢失
