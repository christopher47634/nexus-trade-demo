## 推荐路径

### P4-B Cleanup (推荐优先进入)
**目标:** 清理技术债务，提升稳定性
**范围:** 不新增功能，只修遗留问题

| 任务 | 优先级 | 风险 |

| 修 mobile indices bar 横向溢出 | P2 | 低 |

### P5-A Portfolio Reality (中期)
**目标:** 持仓与资产系统真实感升级

| 任务 | 说明 |
|------|------|
| 持仓数据 | 每只持仓股显示成本价/现价/盈亏/持仓量 |
| 资产计算 | 总资产 = 持仓市值 + 可用资金 |
| 盈亏追踪 | 今日盈亏 / 累计盈亏 |
| 可用资金 | 买入扣减，卖出增加 |
| 模拟账户 | localStorage 存储账户状态 |

**预计工作量:** 2-3 个 session
**前置条件:** P4-B Cleanup 完成

---

### P5-B Market Intelligence (长期)
**目标:** 市场分析能力升级

| 任务 | 说明 |
|------|------|
| 板块轮动 | 板块热度变化趋势 |
| 资金热力 | 全市场资金流向热力图 |
| 风险雷达 | 多维度风险评估面板 |
| 技术指标 | MACD / KDJ / RSI |
| 量化信号 | 买卖信号生成 |

**预计工作量:** 3-5 个 session
**前置条件:** P5-A 完成

## 建议

**当前建议:** 进入 P4-B Cleanup

理由:
1. 移动端溢出影响用户体验，应优先修复
2. 组件重复是技术债务，越早清理越好
3. P4-B 是低风险高收益的改进
4. 为后续 P5-A/B 打下干净基础
